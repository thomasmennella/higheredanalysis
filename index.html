03

How to fill the manual signals

These either have no machine-readable source or are the qualitative tells a faculty member sees first. Each is a short, sourced lookup. Open a section, follow the steps, type the value into the matching cell above.

04

Run the analysis

Scores each filled signal 0 / 1 / 2, then computes a weighted Risk Index per institution. Blank signals are excluded — not counted as “good” — so a partial fill still reads honestly.

Read this as screening, not a verdict. A high index flags an institution worth closer scrutiny; it does not predict closure, and it is not financial or investment advice. Public figures lag — IPEDS finance runs roughly 18 months behind and the most predictive current-cycle signals (this year's yield, retention, discount) live in institutional data before any federal feed carries them. The weighting privileges leading indicators by design, so read the hard-flag counts alongside the index, never the index alone.
